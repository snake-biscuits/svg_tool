# Pixel to Inch Conversion
```
   2px <=> 1/16"
   4px <=> 1/8"
   8px <=> 1/4"
  16px <=> 1/2"
  28px <=> 7/8"
  32px <=> 1"
  38px <=> 1 3/16"
  40px <=> 1 1/4"
  64px <=> 2"
  72px <=> 2 1/4"
  80px <=> 2 1/2"
  96px <=> 3"
 112px <=> 3 1/2"
 ```
